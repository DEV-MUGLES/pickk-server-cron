## Requirements

- [nodejs](https://github.com/nodejs/node) >= 0.12.x

## Getting Started

```shell
$ # 1. install packages
$ npm install

$ # run locally (+ Hot reload)
$ npm run dev

$ # build & run
$ npm run build
$ npm run start

$ # prettier, eslint, jest
$ npm run format
$ npm run lint
$ npm run test
```
